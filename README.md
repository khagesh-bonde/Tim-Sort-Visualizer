# 🚀 Tim Sort Visualizer

An interactive web-based visualization of the **Tim Sort** algorithm built using HTML, CSS, and JavaScript.

This project demonstrates how Tim Sort efficiently combines **Insertion Sort** and **Merge Sort** to optimize sorting performance.

---

## 📌 Features

- 🔄 Step-by-step sorting animation  
- 📊 Real-time counters for Comparisons, Swaps, and Merges  
- ⚡ Adjustable speed control  
- ⏯ Pause, Play, Reset functionality  
- 🎨 Clean and responsive UI  
- 📚 Visual representation of runs and merge phases  

---

## 🧠 How Tim Sort Works

Tim Sort is a hybrid sorting algorithm derived from:

- **Insertion Sort** (for small runs)
- **Merge Sort** (for merging sorted runs)

Steps:
1. The array is divided into small chunks called *runs*.
2. Each run is sorted using Insertion Sort.
3. Sorted runs are merged using the Merge process.
4. Final sorted array is produced efficiently.
