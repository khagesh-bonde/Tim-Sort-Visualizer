Tim Sort

Instructions▼

  • Tim Sort is a hybrid sorting algorithm derived from merge sort and insertion sort
  • Blue bars represent unsorted elements
  • Light green bars show elements being compared or processed
  • Dark green bars are fully sorted elements
  • Tim Sort divides the array into small chunks called "runs" (minimum size: 8)
  • Each run is sorted using insertion sort, then runs are merged together
  • Click "Start" to begin the automatic sorting animation
  • After starting, the button changes to "Next" for step-by-step control
  • Click "Pause" to pause (it changes to "Play" to resume)
  • Click "Reset" to start over with a new random array
  • Adjust the speed using the slider (left = slower, right = faster)
